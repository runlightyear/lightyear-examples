## To Run Lightyear

Install the packages

```shell
npm install
```

Then get your Lightyear credentials

```shell
npx lightyear login
```

When ready, run the Lightyear dev server

```shell
npm run dev
```

## To Run the React App

```shell
cd react-app
```

Install the packages

```shell
npm install
```

Then run the React app

```shell
npm start
```
